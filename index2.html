second paragraph inside div

2. Child selector

First direct child paragraph inside div

second direct child paragraph inside div

Third not a direct child (it nested under section) paragraph inside div

Fourth paragraph inside div (direct child of above div but not of main div[id="child"])

> is used between elements to select as child selector

3. Adjucent-sibling selector

First - Adjucent (neibhour) sibling p element of sibling div

second - not Adjucent (neibhour) sibling p element of sibling div

Third - not Adjucent (neibhour) p element of sibling div

Fourth - Adjucent (neighbour) of above div with same class="A_sibling"

+ is used between elements to select as Adjucent-sibling selector

4. General-sibling selector

First - Adjucent (neibhour) sibling p element of sibling div

second - not Adjuscent (neibhour) sibling p element of sibling div, but as its a sibling of div, this sattement will get selected (as we have used general sibling selector).

Third - not a sibling as well as adjucent (neibhour) sibling p element of sibling div

fourth- not a sibling as well as adjucent (neibhour) sibling p element of sibling div, but A_sibling css will applied here, as it also come under that rule.

Fifth - Adjucent (neibhour) sibling p element of above sibling div[class="G_sibling]

~ is used between elements to select as General-sibling selector
